About Me

My name is Jacob and I am a full-stack developer in the making. For 15 years I worked in the food-service industry making my way up to high-end bartending and management. 2020 seemed like the perfect opportunity to start a new career and training to be a developer with DigitalCrafts was the perfect choice for me.

In addition to being a musician, I am an avid rock climber, yogi, and outdoor adventurer. If it's out there waiting to be explored, I'm here to find it.

ReadMe Book Swap

ReadMe Book Swap is an app for book lovers and new readers alike! RMBS provides a space for connecting with readers who want to trade books with others. Offer up a book you love and share it with friends! RMBS is the tiny library on the corner in your neighborhood if it was an app! Come search and find who has a book that you would like to borrow, barter, or lend.

I was a member of a four student team of developers who used React, Redux, Bootstrap CSS, Sass, and JavaScript to create ReadMe Book Swap.

The Movie Matrix

The Movie Matrix is a mobile-responsive app that allows users to search from thousands of movies and make custom lists of films they have seen or would like to see.

As a member of a three person development team, I was responsible for ensuring a clean, functional user experience and mining data from two separate data sources and tying them together to maximize the information to which our users have access.

The Movie Matrix was built using HTML5, Tailwind CSS, PostgreSQL, Sequelize, AWS, Node.js, JavaScript

Slope Search

Slope Search uses real-time weather and location data to allow users to find the snowy weather nearest them. In addition it shows the closest ski slopes so that the user can plan a winter getaway.

I designed the layout and user experience for Slope Search as well as assisting my team members with data manipulation and rendering.

Slope Search was built using HTML5, CSS, Bootstrap, and JavaScript
